# 安装nginx并使用nginx将hugo静态网站配置到web

# 一、安装nginx

使用命令
```markdown
sudo apt-get install nginx
```
随后使用命令nginx -v查看是否成功

![](/images/jietu22.png)

随后开启nginx服务：
```markdown
service nginx start
```
最后通过ifconfig查看ip地址，然后在浏览器输入ip地址，就可以查到下图所示网页啦

![](/images/jietu23.png)
